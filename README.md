# Copilot Customizations

A presentation on customization options for GitHub Copilot in VS Code.

## Topics Covered

- Custom Instructions (`copilot-instructions.md`)
- File-Scoped Instructions (`*.instructions.md` with `applyTo` globs)
- Prompt Files (`*.prompt.md`)
- Custom Agents (`*.agent.md`)
   - Agent Handoffs & Subagents
- Agent Skills (`SKILL.md`)
- Hooks
- MCP Servers
